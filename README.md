# Title
**Badminton Court Management & Blockchain Loyalty System** *(Hệ thống Quản lý Sân Cầu Lông tích hợp Token Điểm thưởng)*

# Description
This project aims to digitize and optimize the operation of sports complexes while ensuring absolute financial transparency. By adopting a hybrid architecture that combines a robust relational database with Stellar Blockchain (Soroban), the system delivers high-speed transaction processing alongside a trustless environment. The core innovation lies in tokenizing customer loyalty points into digital assets (Badminton Point Token - BPT), preventing database manipulation, and laying the foundation for a broader, decentralized sports ecosystem.

# Features
* **Smart Court Booking:** Advanced conflict resolution for scheduling with dynamic pricing mechanisms based on peak hours and holidays.
* **Inventory & Service Management:** Real-time tracking of sporting goods and beverages, utilizing database compound triggers to maintain strict data integrity without mutating table errors.
* **Automated Payment Integration:** Dynamic VietQR generation for seamless transactions, strictly separating the Invoice (Revenue) and Payment (Cash Flow) lifecycles.
* **Blockchain-Powered Loyalty (Soroban/Rust):** A fully functional smart contract on the Stellar Testnet allowing the Minting, Burning, and Peer-to-Peer Transferring of reward tokens.
* **Role-Based Access Control:** Secure user registration with a default "Customer" role and a strict Admin-approval workflow for "Employee" privileges.
